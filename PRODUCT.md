# Product

## Register

product

## Users

ผู้สร้างคอนเทนต์ภาษาไทยที่ทำงานคนเดียวบนเครื่อง Windows ต้องการตัดช่วงเงียบ สร้างและจัดรูปแบบซับ และเก็บรายละเอียดการตัดต่อไว้ใน timeline โดยไม่ส่งไฟล์ขึ้น cloud

## Product Purpose

โปรแกรมตัดต่อวิดีโอแบบ local-first ที่รวมการตรวจจับช่วงเงียบ การสร้างซับภาษาไทยระดับคำ และ timeline แบบละเอียดไว้ในหน้าจอเดียว ความสำเร็จคือผู้ใช้สามารถนำเข้าวิดีโอ ตรวจแก้จุดตัด แต่งซับ และ export ได้โดยไม่ต้องพึ่ง CapCut

## Brand Personality

แม่นยำ เป็นมืออาชีพ และเข้าถึงง่าย เครื่องมือควรรู้สึกตอบสนองเร็ว น่าเชื่อถือ และไม่ขวางกระบวนการสร้างงาน

## Anti-references

ไม่เลียนแบบแบรนด์หรือทรัพย์สินเฉพาะของ CapCut ไม่ทำหน้าจอแบบ dashboard SaaS ที่เต็มไปด้วยการ์ด ไม่ซ่อนการควบคุม timeline ขั้นสูง และไม่ทำ workflow ที่บังคับใช้ cloud หรือบัญชีผู้ใช้

## Design Principles

- Preview และ timeline เป็นศูนย์กลางของทุกการตัดสินใจ
- Automation ต้องตรวจสอบ แก้ไข และย้อนกลับได้เสมอ
- เริ่มใช้ง่ายแบบ editor สำหรับ creator แต่รองรับความละเอียดแบบ Premiere Pro
- ทุกสถานะสำคัญต้องมองเห็นได้จากสี รูปร่าง และข้อความ ไม่พึ่งสีเพียงอย่างเดียว
- เก็บสื่อ โมเดล และข้อมูลโปรเจกต์ไว้ในเครื่อง

## Accessibility & Inclusion

รองรับ WCAG AA, keyboard focus ที่ชัดเจน, reduced motion, contrast ที่อ่านง่าย และข้อความภาษาไทยที่ไม่ถูกตัดหรือบีบจนอ่านยาก
